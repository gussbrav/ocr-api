API OCR - PDF a Texto

🚀 API Lista

Esta API convierte documentos PDF a texto usando OCR inteligente. Detecta automáticamente si el PDF contiene texto extraíble o necesita procesamiento de imágenes.

📋 Endpoints Disponibles

GET /health

Verificar el estado del servicio

curl -X GET http://localhost:5000/health
POST /ocr/process

Procesar un archivo PDF individual

Parámetros:

  • file - Archivo PDF (required)
  • language - Idioma (spa, eng, etc.) - Opcional
  • format - Formato de respuesta (txt, json) - Opcional
curl -X POST http://localhost:5000/ocr/process \
  -F "file=@documento.pdf" \
  -F "language=spa" \
  -F "format=json"
POST /ocr/batch

Procesar múltiples archivos PDF (devuelve ZIP)

Parámetros:

  • files[] - Múltiples archivos PDF (required)
  • language - Idioma - Opcional
curl -X POST http://localhost:5000/ocr/batch \
  -F "files[]=@doc1.pdf" \
  -F "files[]=@doc2.pdf" \
  -F "language=spa" \
  --output results.zip

🧪 Probar API

Usa Postman, curl, o integra directamente con n8n usando los ejemplos de arriba.

Ejemplo de respuesta JSON:

{
  "success": true,
  "filename": "documento.pdf",
  "pages_processed": 3,
  "processing_time": 15.7,
  "method": "ocr",
  "language": "spa",
  "text": "Texto extraído del PDF...",
  "timestamp": "2025-08-31T15:30:45"
}

🔧 Para uso con n8n

Configura un nodo HTTP Request en n8n con:

  • Method: POST
  • URL: http://tu-dominio.com/ocr/process
  • Body Type: Form-Data
  • Parameters: file, language, format
